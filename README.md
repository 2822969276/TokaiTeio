# TokaiTeio

## 安装步骤

1. 安装python

    MkDocs 需要安装 Python 环境，MkDocs 支持 Python 2.6, 2.7, 3.3 和 3.4。可通过 pip 安装。


    [Python3 官网](https://www.python.org/)

    [download python-3.11.1 for windows](https://www.python.org/ftp/python/3.11.1/python-3.11.1-amd64.exe)


2. 安装mkdocs

``` shell
pip install mkdocs
```
